# 💇‍♀️ Hair Day - Sistema de Agendamento

## 📌 Sobre o projeto

O **Hair Day** é uma interface de sistema de agendamento para salão de beleza, desenvolvida com foco em organização, usabilidade e design moderno.

A aplicação permite selecionar datas, horários disponíveis e cadastrar clientes, além de visualizar os agendamentos organizados por período do dia.

---

## 🎨 Tecnologias utilizadas

* HTML5
* CSS3

---

## ✨ Funcionalidades

* Seleção de data para agendamento
* Escolha de horários disponíveis
* Identificação visual de horários indisponíveis
* Cadastro do nome do cliente
* Listagem de agendamentos por período:

  * Manhã
  * Tarde
  * Noite
* Interface organizada e intuitiva
* Ícones visuais para melhor experiência do usuário

---

## 🧠 Estrutura da aplicação

### 📅 Formulário de agendamento

* Input de data
* Lista de horários interativos
* Campo para nome do cliente
* Botão para realizar agendamento

### 📋 Painel de agendamentos

Os agendamentos são exibidos separadamente por período:

* 🌅 Manhã (09h - 12h)
* ☀️ Tarde (13h - 18h)
* 🌙 Noite (19h - 21h)

Cada item exibe:

* Horário
* Nome do cliente
* Opção de cancelamento
